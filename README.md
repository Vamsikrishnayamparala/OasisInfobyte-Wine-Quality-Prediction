# OasisInfobyte-Wine-Quality-Prediction
This project predicts wine quality using machine learning algorithms and chemical characteristics of wine.

## Project Overview
This project predicts wine quality using machine learning algorithms and chemical characteristics of wine.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Random Forest
- SGD Classifier
- SVC

## Project Tasks
- Data Exploration
- Data Cleaning
- Feature Selection
- Model Training
- Model Comparison

## Key Findings
- Random Forest achieved the highest accuracy of approximately 70%.
- Chemical attributes strongly influence wine quality.
- Model comparison identified the best-performing algorithm.

## Files
- Wine_Quality_Prediction.ipynb

## Conclusion
The project demonstrates classification techniques for quality prediction.
